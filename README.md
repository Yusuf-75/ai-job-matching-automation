AI-Driven Job Market Intelligence & Automation
Überblick
Dieses Projekt automatisiert meine Jobsuche. Ein LinkedIn-Scraper findet Stellen, eine KI (Llama 3.3) bewertet sie, und ich bekomme die besten Treffer direkt auf mein Handy.

Technische Highlights
* Automatisierung: n8n Workflow auf einer Azure VM.
* KI-Logik: Semantische Analyse mit Groq LLM (Llama 3.3).
* Datenbank: Speicherung und Duplikatsprüfung in PostgreSQL.
* Monitoring: Automatischer Fehler-Alarm via Telegram-Bot.

Warum das wichtig ist
Es zeigt, wie man Business-Prozesse (Jobsuche) durch moderne IT (KI & Cloud) effizienter macht.
